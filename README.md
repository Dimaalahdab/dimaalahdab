## 🚀 Featured Projects

- **[Syria_Remembers](https://github.com/dimaalahdab/Syria_Remembers)** — A database management system for documenting massacres from the Syrian war. Built with user roles (Admin/Researcher), it allows secure management of massacre records by date, type, location, and more. Built in Java.

- **[SyrianRevolutionHeroes](https://github.com/dimaalahdab/SyrianRevolutionHeroes)** — A web page honoring the bravery of Syrian heroes who rose against oppression. It shares their stories and sacrifices, built using HTML, CSS, and JavaScript to preserve their legacy.

- **[WeatherWise](https://github.com/fatima448/weatherwise)** 🏆 *Ranked 14th — Anadolu Hackathon 2026* — An AI-powered weather app that converts raw forecasts into instant, actionable decisions. Instead of showing numbers, it tells you what to *do* with them — what to wear, whether to grab an umbrella, which activities suit the weather, and whether your scheduled tasks will clash with the forecast at that exact hour. Built with React 19 + FastAPI backend running **9 scikit-learn ML models** (Random Forest, Gradient Boosting) for clothing, umbrella, activity, and safety predictions — plus Gemini 2.0 Flash for task conflict classification and LLaMA 3.3 70B for the chat assistant. Built from scratch in a few days with Fatima.

- **[DimaBeautyCenter](https://github.com/dimaalahdab/DimaBeautyCenter)** *(University Project)* — A full-stack beauty center management system. The admin panel handles appointments with automatic conflict detection, client records, employee profiles, and a service catalog with pricing. Built with **Java + Spring Boot** on the backend, **REST APIs** (GET, POST, PUT, DELETE) connecting to a **MySQL** database via JPA/Hibernate, and a vanilla **HTML, CSS, JavaScript** frontend.

- **[DimaLanguageAcademy](https://github.com/dimaalahdab/DimaLanguageAcademy)** *(University Project)* — A web-based student management system built for a language academy that previously managed everything manually with papers. Admins can add, view, edit, and delete student records, search by name, and browse with pagination. Built with **PHP** and **MySQL** on the backend, and **HTML, CSS, JavaScript** on the frontend, with session-based authentication protecting every page.

- **[Paw & Love Vet Clinic](https://github.com/dimaalahdab/Veterinary-Clinic)** *(University Project)* — A multi-page veterinary clinic website built with pure HTML and CSS. Includes a home page, services page, team profiles, patient stories, and a contact page — all connected through a shared navigation bar and consistent styling.

- **[cyberbullying_detection_system](https://github.com/dimaalahdab/cyberbullying_detection_system)** — An ML-based system for detecting cyberbullying. Built with Python and Jupyter Notebook.
